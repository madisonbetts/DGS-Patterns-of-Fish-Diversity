Called genotypes file info

calledgenos_minInd50_95p.csv
4396 individuals
473 SNPs
(all SNPs where max. 50% of the individuals are missing data)
-This data set was used to calculate Fst values and heterozygosity

Each row is a SNP
Each individual has 3 columns:
-the sample_ID is listed over the middle column in letter form:
  AGTC - combination of two per individual
  NN = no data
-the previous column (sample_ID - 1) is in numeric form:
  -1 = no data
  0 = homozygous for the major allele
  1 = heterozygous
  2 = homozygous for the minor allele
-the column following the middle column (sample_ID + 1) is the posterior probability of the most likely (called) genotype


Individuals are identified by "sample_ID", which is a combination of their capture location and location on a genetic plate
"Sample_ID" can be merged with further information about each fish in the "plate_maps_omy5genotypes.csv" file
