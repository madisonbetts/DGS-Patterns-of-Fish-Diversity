


Description of the data in this Dryad package:

This data is for the following manuscript:

Landscape Genomics Modeling Suggest Climate Variables Explain Neutral and Adaptive Variation in Replicate Metapopulations - 

Authors - Brian K. Hand, Clint C. Muhlfeld, Alisa A. Wade, Ryan P. Kovach, Diane C. Whited, Shawn R. Narum, Andrew P. Matala, Michael W. Ackerman, Brittany A. Garner, John S. Kimball, Jack A. Stanford, Gordon Luikart

For publication in Molecular Ecology.


Pairwise_Distance_Matrices: This folder contains the pairwise Fst matrices used in DISTLM for both neutral and adaptve loci. 

SH_genepop_files: This folder contains genepop files for both netural and adaptivel loci for each of five steelhead metapopulations. 

SH_geste_input_files: This folder cointains the necessary environmental and genetic data input files to replicate runs of the GESTE program from the manuscript. 

SH_Environmental_Data.xlsx: This excel file contains all the calculated envrionmental variables for each of 79 steelhead popultions along with the population name, the pop id corresponding to the same identifiers used in the genetic file below, the MPG where samples were collected, the years of sampling, and the population DPS unit, followed by environemtnal data. 

SH_Raw_Genetic.xlsx: This excel file contains all genotypes for neutral and adaptivel loci in a genelex formated file. The genetic data in this Dryad package is a subset of the data archived in http://dx.doi.org/10.1111/eva.12174.
